# ARS Auto - Guide Utilisateur

Guide utilisateur interactif pour le simulateur de tarification automobile d'ARS.

## 🚀 Démo en ligne

Visitez le guide : [https://khiari-mohamed.github.io/ars_simulator_user_guide/guide-ARS.html](https://khiari-mohamed.github.io/ars_simulator_user_guide/guide-ARS.html)

## 📋 Description

Plateforme web interactive permettant aux utilisateurs de :
- Créer des simulations d'assurance automobile
- Comparer les offres de différentes compagnies
- Générer et télécharger des devis
- Souscrire en ligne via paiement sécurisé Flouci

## 🛠️ Technologies

- HTML5
- CSS3 (Design moderne et responsive)
- JavaScript (Vanilla)
- Google Fonts (Playfair Display, DM Sans)

## 🌐 Déploiement sur GitHub Pages

1. **Créer le repository sur GitHub**
2. **Pousser le code**
3. **Activer GitHub Pages** :
   - Allez dans Settings > Pages
   - Source : Deploy from a branch
   - Branch : main / root
   - Cliquez sur Save

Le site sera disponible à : `https://khiari-mohamed.github.io/ars_simulator_user_guide/guide-ARS.html`

## 💻 Développement local

Ouvrez simplement `guide-ARS.html` dans votre navigateur ou utilisez un serveur local :

```bash
# Avec Python
python -m http.server 8000

# Avec Node.js
npx serve

# Avec PHP
php -S localhost:8000
```

Puis visitez : `http://localhost:8000/guide-ARS.html`
